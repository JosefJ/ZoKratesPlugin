ZoKrates Plugin for Remix IDE
==================================

This is work in progress!! 

Getting Started
---------------
Just run
`sudo docker run -p 127.0.0.1:8080:8080 -ti zkplug:latest /bin/bash`

`cd ZoKratesPlugn`
`npm start`


Plugin definition
-----------------

```
{
"title": "ZoKrates",
"url": "http://127.0.0.1:8080"
}
```
